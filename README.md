# Mini Entulho Tambores

Site PWA para a empresa Mini Entulho Tambores, com base em Fazenda Rio Grande.

## Arquivos principais

- `index.html`: pagina principal do site.
- `styles.css`: identidade visual e responsividade.
- `app.js`: calculadora simples e link dinamico para WhatsApp.
- `manifest.webmanifest`: configuracao de instalacao como PWA.
- `sw.js`: cache offline basico.

## Publicacao

Este site e estatico e pode ser publicado na Vercel direto pelo GitHub.

1. Crie um repositorio no GitHub.
2. Envie este projeto para o repositorio.
3. Na Vercel, importe o repositorio.
4. Use as configuracoes padrao, sem comando de build.
5. A cada `git push`, a Vercel publica automaticamente uma nova versao.
